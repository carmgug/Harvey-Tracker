# Project Overview: Hurricane Harvey Tweet Analysis Platform  

This project focuses on the development of a comprehensive platform for analyzing tweets related to Hurricane Harvey. The platform leverages **Apache Spark** for backend data processing and **Angular** for a dynamic and interactive frontend.  

---

## **Key Features**

1. **Backend (Spark)**  
   - Efficiently processes large volumes of tweet data in real-time or batch mode.  
   - Implements advanced filtering, sentiment analysis, and keyword extraction specific to Hurricane Harvey discussions.  
   - Provides APIs to deliver processed data to the frontend.  

2. **Frontend (Angular)**  
   - Offers a user-friendly interface for accessing and exploring data insights.  
   - Includes tools for sorting, filtering, and detailed analysis of tweet content.  

3. **Geographical Visualization**  
   - Displays the geographical distribution of tweets, highlighting areas with the highest activity.  
   - Supports zoom and drill-down functionality for granular insights.  

---

## **Objectives**  
- Build an end-to-end platform to analyze the public's perception and response to Hurricane Harvey via Twitter.  
- Facilitate intuitive visualization and exploration of geo-tagged data.
- Enable detailed sentiment and content analysis to support disaster response and decision-making.  

This project serves as a robust tool for social media analysis in the context of natural disasters, with potential applications in other domains.
